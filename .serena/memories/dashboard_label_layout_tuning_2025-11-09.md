# 柱状图标签布局优化（2025-11-09）
- 背景：仪表盘多张柱状图（收入、单耗、标煤等）的数据标签存在重叠或离柱体过远的问题。
- 操作：在 `frontend/src/daily_report_25_26/pages/DashBoard.vue` 中为相关 ECharts 系列统一设置 `distance: 6`，并采用 `labelLayout: { moveOverlap: 'shiftY' }` 通过垂直位移自动避让；同时移除旧的手动 offset 与隐藏策略，使所有标签都能显示。
- 影响：标签不再互相遮挡且保持贴近柱体，数据可读性提升；图表数据与配色逻辑未变。
- 回滚：恢复标签配置至修改前状态即可撤销该自动避让策略。
